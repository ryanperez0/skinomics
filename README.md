# Skinomics: Quantitative Analysis of CS2 Digital Assets
A project dedicated to applying financial modeling and data-driven strategies to the Counter-Strike 2 market.

## Project Objectives
* **Market Modeling**: Tracking price volatility across high-liquidity skins.
* **Expected Value Analysis**: Calculating the ROI on "trade-up" contracts.
* **Arbitrage Detection**: Identifying price discrepancies between third-party marketplaces and the Steam Community Market.

## Tech Stack
* **Language**: Python (Pandas)
* **Database**: SQL (SQLite)
* **Visualization**: Plotly
